## Clone git repository and install packages

    git clone https://github.com/OluwaferanmiLove/ajeoba.git
    cd ajeoba

Then install npm packages with yarn

    yarn install

Then install expo packages with

    npx expo install

## Run application

Start metro server

    npx expo start

Open Camera and scan QR code to run
